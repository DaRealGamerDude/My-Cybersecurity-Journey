# Web Hacking (Module 08)

This module builds foundational knowledge of **web application security**, covering how web apps function, how client-side and backend logic works, and how attackers exploit weaknesses using real tools like Burp Suite.

It focuses on understanding:
- Web communication (HTTP/HTTPS)
- Client-side scripting (JavaScript)
- Backend databases (SQL)
- Traffic interception and manipulation

---

# Module Structure

### 01 — Web Application Basics  
→ [`01-Web-Application-Basics`](./01-Web-Application-Basics)

Covers how web applications work, including **HTTP requests/responses, methods, headers, status codes, and URLs** — forming the foundation for all web-based attacks.

---

### 02 — JavaScript Essentials  
→ [`02-Javascript-Essentials`](./02-Javascript-Essentials)

Introduces **client-side scripting**, how JavaScript interacts with HTML, and how attackers abuse features like **dialogs, control flow, and client-side validation**.

---

### 03 — SQL Fundamentals  
→ [`03-SQL-Fundamentals`](./03-SQL-Fundamentals)

Builds understanding of **databases and SQL queries**, including CRUD operations, filtering, and functions — essential for understanding **SQL Injection and backend data flow**.

---

### 04 — Burp Suite: The Basics  
→ [`04-Burp-Suite-The-Basics`](./04-Burp-Suite-The-Basics)

Introduces **Burp Suite**, the industry-standard web testing tool, focusing on **proxying, intercepting traffic, modifying requests, and basic attack workflows (e.g., XSS)**.

---

# Key Skills Developed

- Understanding web architecture  
- Interpreting HTTP traffic  
- Identifying client-side vs server-side logic  
- Working with databases and queries  
- Intercepting and modifying web requests  
- Recognizing common web vulnerabilities
