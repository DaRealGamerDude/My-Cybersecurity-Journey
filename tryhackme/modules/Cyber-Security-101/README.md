# Cyber Security 101 — TryHackMe Journey

This section documents my structured progression through the **TryHackMe Cyber Security 101 pathway**, covering both **Offensive and Defensive security fundamentals**.

The goal of this journey is to build a strong foundation for:

**SOC Analyst → Threat Detection → DFIR (Digital Forensics & Incident Response)**

---

# Overview

This pathway is designed to take a learner from **zero to foundational cybersecurity knowledge**, covering:

- Operating systems (Linux, Windows, AD)  
- Networking fundamentals  
- Cryptography  
- Exploitation techniques  
- Web security  
- Defensive security (SOC, logs, forensics)  
- Security tools and solutions  
- OWASP Top 10 vulnerabilities
